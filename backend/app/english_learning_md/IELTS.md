# IELTS 备考攻略（2026版）

> IELTS（International English Language Testing System，雅思）由British Council（英国文化教育协会）、IDP和剑桥大学考试委员会联合主办。2026年核心题型和评分体系不变，但纸笔考试将于2026年6月底全面退出，全球转为机考。本指南以**IELTS Academic（学术类）**为主。


## 一、考试概述

| 项目 | 详情 |
|------|------|
| 考试全称 | IELTS (International English Language Testing System) |
| 两种类型 | Academic（学术类：留学）、General Training（培训类：移民/工作） |
| 总时长 | 2小时45分钟 |
| 科目数 | 4个：听力、阅读、写作、口语 |
| 评分体系 | 0-9分等级制（0.5分为一档），总分为四科均分（四舍五入到最近0.5分） |
| 及格线 | 无官方及格线，多数大学要求6.5-7.0 |
| 有效期 | 2年 |
| 考试费用 | 中国大陆约2170元人民币（机考） |

### Academic 与 General Training 的区别

| 项目 | Academic（学术类） | General Training（培训类） |
|------|-------------------|--------------------------|
| 用途 | 本科/硕士/博士留学 | 移民、工作、中学留学 |
| 听力 | 相同 | 相同 |
| 口语 | 相同 | 相同 |
| 阅读 | 学术文章（期刊、专著） | 日常+工作+学术混合 |
| 写作Task 1 | 图表描述（折线图、柱状图、饼图、流程图等） | 书信写作（求职信、投诉信等） |
| 写作Task 2 | 相同（学术议论文） | 相同 |

> **核心公式**：听力 + 阅读 = 80题客观题，是提分最快的两科。写作 + 口语 = 主观评分，需要长期积累。


## 二、题型与分值详解

### 2.1 整体结构

| 科目 | 时长 | 题量 | 分值 | 核心考察 |
|------|------|------|------|---------|
| 听力 Listening | 30分钟 | 40题 | 0-9分 | 信息提取、细节理解、逻辑推理 |
| 阅读 Reading | 60分钟 | 40题 | 0-9分 | 主旨理解、细节定位、观点判断 |
| 写作 Writing | 60分钟 | 2篇 | 0-9分 | 数据描述/书信、议论文写作 |
| 口语 Speaking | 11-14分钟 | 3部分 | 0-9分 | 流利度、词汇、语法、发音 |

> **注意**：雅思没有负分机制，答错不扣分。务必每道题都作答，不要留空白。


### 2.2 听力（Listening）—— 30分钟，40题

听力部分对Academic和General Training完全相同。录音只播放一次，包含多种英语口音（英式、美式、澳式等）。

#### Part 1：日常对话（10题）

- **场景**：两人间的日常社交对话（如预订旅行、租房咨询等）。
- **难度**：最简单，用于热身。
- **技巧**：边听边填，注意拼写和数字。

#### Part 2：日常独白（10题）

- **场景**：一人独白，介绍某个设施、活动或服务（如图书馆导览、社区活动介绍）。
- **难度**：中等。
- **技巧**：提前读题，预判答案类型。

#### Part 3：学术讨论（10题）

- **场景**：2-3人的学术讨论（如学生与导师讨论论文、小组讨论课题）。
- **难度**：较高，说话人切换频繁。
- **技巧**：区分不同说话人的观点，注意转折（but, however, actually）。

#### Part 4：学术讲座（10题）

- **场景**：一人学术讲座（如大学课堂）。
- **难度**：最高，学术词汇多。
- **技巧**：快速笔记，抓讲座结构（引言-论点-例证-结论）。

#### 听力常见题型

| 题型 | 说明 |
|------|------|
| **填空题**（表格/笔记/摘要/句子填空） | 根据录音填写缺失信息，注意字数限制 |
| **选择题** | 单选或多选 |
| **匹配题** | 将信息与选项配对 |
| **地图/流程图标注** | 在图上标注位置或流程步骤 |
| **简答题** | 用不超过规定字数回答问题 |

### 2.3 阅读（Reading, Academic）—— 60分钟，40题

3篇长文，总词数2150-2750词，选自学术期刊、书籍、杂志和报纸。题材涵盖科学、技术、历史、心理学、社会议题等。

#### 11种官方题型

| 序号 | 题型 | 说明 |
|:----:|------|------|
| 1 | **选择题 (Multiple Choice)** | 单选（四选一）或多选（五选二/七选三） |
| 2 | **判断题 True/False/Not Given** | 判断陈述与原文信息是否一致 |
| 3 | **观点判断 Yes/No/Not Given** | 判断陈述是否符合作者观点 |
| 4 | **段落信息匹配 (Matching Information)** | 在指定段落中定位特定信息 |
| 5 | **段落标题匹配 (Matching Headings)** | 为段落选择最合适的标题 |
| 6 | **特征匹配 (Matching Features)** | 将信息与特定的人/事物/理论配对 |
| 7 | **句子结尾匹配 (Matching Sentence Endings)** | 将句子前半部分与正确的后半部分配对 |
| 8 | **句子填空 (Sentence Completion)** | 用原文词汇填空完成句子 |
| 9 | **摘要/表格/流程图填空** | 在摘要或图表中填入缺失信息 |
| 10 | **图表标注 (Diagram Label Completion)** | 在图表上标注信息 |
| 11 | **简答题 (Short-Answer Questions)** | 用原文信息简短回答问题 |

**2026年趋势变化**：
- Matching Headings题出现频率下降，**Matching Sentence Endings大幅增加**——必须理解完整句子的语法和逻辑。
- 不能再仅靠关键词定位"hack"答案，需要真正理解句意。

**核心技巧**：
- **先扫题再读文**——知道要找什么比盲目通读效率高100倍。
- **True/False/Not Given是重灾区**——"Not Given"指原文完全未提及，不等于"推理后觉得不对"。
- **严格遵守字数限制**——题目要求"NO MORE THAN TWO WORDS"就绝对不能写三个词。

### 2.4 写作（Writing, Academic）—— 60分钟，2篇

Task 2权重是Task 1的**两倍**。

#### Task 1：图表描述（20分钟，150词以上）

- **形式**：描述一个或多个图表（折线图、柱状图、饼图、表格、流程图、地图变迁等）。
- **2026趋势**：**混合图表**（如饼图+表格）出现频率增加，需要综合不同数据源进行分析。
- **评分重点**：
  - 准确描述数据趋势和关键特征。
  - 进行合理的比较和总结。
  - 不要加入个人观点。
- **技巧**：
  - 开头改写题目（不要照抄）。
  - 第二段写总体趋势/最显著特征（Overview），这一段决定你能否上6分。
  - 后续段落分组描述细节数据。

#### Task 2：议论文写作（40分钟，250词以上）

- **形式**：就某个话题发表观点、讨论问题或提出解决方案。
- **常见题型**：
  - Agree/Disagree（同意或不同意）
  - Discuss Both Views（讨论两方观点）
  - Problem/Solution（问题与解决方案）
  - Advantages/Disadvantages（优缺点分析）
- **评分重点**：
  - 回应题目要求（审题！不要偏题）。
  - 逻辑清晰、论证充分。
  - 词汇丰富且准确。
  - 语法多样且正确。

**2026严厉警告**：考官经过重新培训，会**严惩套用模板的作文**。如果使用背诵的万能开头、万能过渡句或万能结尾，Task Response可能被封顶在4.0分。必须针对具体题目写作。

#### 写作评分四维度（Task 1和Task 2相同）

| 维度 | 占比 | 说明 |
|------|:----:|------|
| **Task Achievement/Response** | 25% | 是否完成任务要求、论点是否清晰 |
| **Coherence and Cohesion** | 25% | 逻辑组织、段落衔接、连接词使用 |
| **Lexical Resource** | 25% | 词汇丰富度和准确度 |
| **Grammatical Range and Accuracy** | 25% | 语法多样性和正确率 |


### 2.5 口语（Speaking）—— 11-14分钟，面对面

口语对Academic和General Training完全相同。与真人考官面对面交流（部分考点已改为**视频通话VCS模式**——仍是真人考官，但通过屏幕交流）。

#### Part 1：日常问答（4-5分钟）

- **形式**：考官就日常话题提问（家乡、工作/学习、爱好、日常生活等）。
- **目的**：热身，考察基本交流能力。
- **技巧**：回答2-3句即可，不要太短（一个词）也不要太长（讲故事）。自然即可。

#### Part 2：个人独白（3-4分钟）

- **形式**：拿到话题卡（cue card），有**1分钟准备时间**（可做笔记），然后**说2分钟**。
- **话题卡内容**：Describe a person / place / event / experience ...（描述某人/某地/某事/某种经历）。
- **技巧**：
  - 1分钟准备时快速列提纲（关键词，不是完整句子）。
  - 围绕Who/What/When/Where/Why/How展开。
  - 要说满2分钟，考官可能在中途打断，这是正常的。

#### Part 3：深入讨论（4-5分钟）

- **形式**：考官就Part 2话题的延伸方向提问，问题更抽象、更有深度。
- **目的**：考察分析能力、观点表达、逻辑论证。
- **技巧**：
  - 使用"观点+原因+举例"结构。
  - 不要只说 "I think..." 开头，使用更多样的表达方式。
  - 可以适当展示让步和对比（"While some might argue... I believe..."）。

#### 口语评分四维度（各占25%）

| 维度 | 说明 |
|------|------|
| **Fluency and Coherence** | 流利度和连贯性——能否持续表达而不频繁停顿 |
| **Lexical Resource** | 词汇资源——是否使用丰富且准确的词汇 |
| **Grammatical Range and Accuracy** | 语法多样性和准确度 |
| **Pronunciation** | 发音清晰度和自然度——不要求完美口音，但要清晰可懂 |


## 三、评分体系详解

### 3.1 等级量表（Band Scale）

| 等级 | 水平 | 描述 |
|:----:|------|------|
| 9 | Expert User | 完全掌握，使用准确、流利、恰当 |
| 8 | Very Good User | 偶有不准确，能很好地处理复杂论述 |
| 7 | Good User | 能力强，偶有失误，总体掌握良好 |
| 6 | Competent User | 尽管有不准确和误解，能较有效使用语言 |
| 5 | Modest User | 部分掌握，能在多数场景下理解大意 |
| 4 | Limited User | 基本能力仅限于熟悉场景 |
| 3 | Extremely Limited | 仅能在非常熟悉的场景下沟通 |
| 2 | Intermittent User | 极难用英语沟通 |
| 1 | Non User | 几乎无法使用英语 |
| 0 | Did Not Attempt | 未参加考试 |

### 3.2 总分计算

总分 = 四科成绩的算术平均值，四舍五入到最近的0.5分。

**示例**：听力7.0 + 阅读6.5 + 写作6.0 + 口语6.5 = 26.0 ÷ 4 = 6.5

### 3.3 各大学分数要求参考

| 学校类型 | 总分要求 | 单科要求 |
|---------|:-------:|---------|
| 顶尖名校（牛津、剑桥等） | 7.0-7.5 | 各科不低于6.5-7.0 |
| 优秀大学 | 6.5-7.0 | 各科不低于6.0 |
| 普通大学 | 6.0-6.5 | 各科不低于5.5 |

> **重要**：很多大学对写作和口语有单科最低分要求（通常不低于6.0或6.5），即使总分达标也可能因单科不够被拒。


## 四、2026年重要变化

### 4.1 全面转向机考

2026年6月27日起，全球绝大多数考点**全面停止纸笔考试**，转为计算机机考。
- 听力、阅读、写作在电脑上完成。
- 口语仍为真人考官交流（部分考点改为视频通话VCS模式）。
- 出分时间从纸笔考的13天缩短至**1-5天**。

### 4.2 单科重考（One Skill Retake, OSR）

如果某一科成绩不理想，可以在**原考试60天内只重考那一科**，不需要重新考全部四科。
- 仅限**机考**考生使用。
- 新成绩与其他三科原成绩组合生成新的总分。
- 是2026年最重要的利好政策——某科发挥失常不必全部重来。

### 4.3 反模板力度加大

写作考官经过2025年重新培训，对**模板化写作**的惩罚力度大幅加强：
- Task 2大量使用背诵的万能模板 → Task Response可能被封顶4.0。
- 需要针对题目的具体内容进行个性化写作。

### 4.4 题型微调

- 听力：题目顺序更灵活，如地图标注题可能出现在更靠前的位置。
- 阅读：Matching Sentence Endings题大幅增加，Matching Headings减少。
- 口语VCS模式：Part 2话题卡显示在屏幕上而非纸上，需适应数字设备阅读。


## 五、分阶段备考方法

建议备考周期**2-4个月**。

### 第一阶段：基础摸底（第1-2周）

- 做一套完整的Cambridge IELTS模考（推荐Cambridge IELTS 18-20），评估当前水平。
- 明确目标分数和薄弱科目。
- 制定个性化备考计划。

### 第二阶段：分科强化（第3-8周）

**听力**：
- 每天精听30分钟——先盲听，再对照原文精听，最后跟读。
- 重点训练**Part 3和Part 4**（分值与Part 1、2相同但难度大得多）。
- 练习边听边写的填空能力——注意拼写、单复数、大小写。
- 听BBC、TED、学术讲座适应不同口音。

**阅读**：
- 每天限时做一篇阅读（20分钟/篇），训练速度和准确性。
- 重点攻克**T/F/NG判断题**和**Matching Sentence Endings**。
- 学习"扫读(scanning)+略读(skimming)"技巧——先读题干，再定位原文。
- 积累学术高频词汇（Academic Word List）。

**写作**：
- Task 1：每周练2篇图表描述，重点练**Overview段**（不写Overview基本上不了6分）。
- Task 2：每周练2篇议论文，**绝对不要背模板**。
- 按四维度自评或找老师批改（Task Response > Coherence > Lexical > Grammar）。
- 积累同义替换词和高级句式（定语从句、倒装句、强调句等）。

**口语**：
- 每天练Part 1日常问答——对着镜子说或录音回听。
- 每周练3-4个Part 2话题——严格限时1分钟准备+2分钟独白。
- Part 3练习"深入论证"——不是简单回答Yes/No，要给理由和例子。
- 发音：不追求完美口音，但要保证**清晰可懂**。

### 第三阶段：套题模考+查缺补漏（第9-12周）

- 每周做2套完整模考，严格计时。
- 写作找老师批改或使用可靠的AI批改工具。
- 口语找练习搭档或外教模拟考试。
- 分析错题规律，针对性突破薄弱环节。

### 第四阶段：考前冲刺（最后1-2周）

- 重做之前的错题。
- 写作背诵高分范文的亮点句式（不是背全文模板）。
- 口语整理常考话题的答题思路。
- 调整作息，保持状态。

## 七、雅思 vs 托福 如何选择？

| 维度 | IELTS | TOEFL iBT (2026) |
|------|-------|-----------------|
| 口语形式 | 真人考官面对面 | 对电脑录音（AI评分） |
| 写作形式 | 议论文+图表描述 | 造句+邮件+学术讨论 |
| 评分体系 | 0-9分 | 1-6分 |
| 考试时长 | 2小时45分钟 | 约100分钟 |
| 自适应 | 否 | 是（阅读和听力） |
| 适合地区 | 英国、澳洲、加拿大优先 | 美国优先 |
| 单科重考 | 支持（OSR） | 不支持 |

**选择建议**：
- 去英国/澳洲/加拿大 → 优先考雅思。
- 去美国 → 优先考托福（但多数美国大学同时接受雅思）。
- 害怕面对面口语 → 考托福（对电脑录音）。
- 不擅长长时间高强度集中 → 考托福（100分钟 vs 165分钟）。
- 某科发挥容易不稳定 → 考雅思（支持单科重考）。


## 八、考试特别提醒

1. **答题不要留空白**——没有负分，不答必丢分，猜了还有机会。
2. **听力/阅读填空题注意细节**——大小写、单复数、连字符都可能影响得分。
3. **写作字数是硬性要求**——Task 1低于150词、Task 2低于250词会被扣分。
4. **口语不要背答案**——考官经验丰富，一听就知道你在背。自然表达最重要。
5. **熟悉机考界面**——2026年全面机考，提前在官方练习平台适应电脑答题。
6. **善用单科重考（OSR）**——如果某科差0.5分，不必全部重来。
7. **写作反模板**——2026年考官对模板化答案零容忍，必须针对具体题目写作。
