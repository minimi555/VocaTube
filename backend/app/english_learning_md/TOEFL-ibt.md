# 新托福 TOEFL iBT 备考攻略（2026年1月改版）

> 2026年1月21日，ETS对托福iBT进行了有史以来最大规模改版。考试时长缩短至约100分钟，引入自适应机制，评分体系从0-120分改为1.0-6.0等级制，并新增多种贴近真实生活的题型。本文仅针对2026年新版托福。


## 一、考试概述

| 项目 | 详情 |
|------|------|
| 考试全称 | TOEFL iBT (Test of English as a Foreign Language, Internet-Based Test) |
| 主办方 | ETS (Educational Testing Service) |
| 总时长 | 约100分钟（无休息） |
| 科目数 | 4个：阅读、听力、口语、写作 |
| 考试顺序 | 固定：阅读 → 听力 → 口语 → 写作 |
| 评分体系 | 1.0-6.0等级制（0.5分为一档），总分为四科均分 |
| CEFR对齐 | 直接标注在成绩单上（A1-C2） |
| 自适应机制 | 阅读和听力采用多阶段自适应（MST） |
| 出分速度 | 72小时内出分；阅读和听力可在考场立即查看非正式分数 |
| 考试费用 | 因地区而异，中国大陆约2100元人民币 |

**核心变化（对比旧版）**：

| 特征 | 旧版托福 | 2026新版托福 |
|------|---------|-------------|
| 总时长 | 约2小时 | 约100分钟 |
| 评分 | 0-120（四科各0-30，求和） | 1.0-6.0（四科各1-6，求均分） |
| 自适应 | 无 | 阅读和听力自适应 |
| 综合题 | 有（阅读+听力→口语/写作） | 无 |
| 独立写作 | 有（长篇议论文） | 无 |
| 考试顺序 | 不固定 | 固定 |
| 内容风格 | 100%学术 | 学术+日常生活混合 |
| 新增题型 | — | Complete the Words、Listen and Repeat、Build a Sentence |


## 二、题型与分值详解

### 2.1 整体结构

| 科目 | 时长 | 题量 | 分值 | 自适应 | 核心考察 |
|------|------|------|------|--------|---------|
| 阅读 Reading | ~30分钟 | 35-48题 | 1.0-6.0 | 是（两模块） | 词汇语境、日常阅读、学术理解 |
| 听力 Listening | ~29分钟 | 多题 | 1.0-6.0 | 是（两模块） | 语用理解、对话、通知、学术讲座 |
| 口语 Speaking | ~8分钟 | 11题 | 1.0-6.0 | 否 | 复述、即兴面试 |
| 写作 Writing | ~23分钟 | 12题+2篇 | 1.0-6.0 | 否 | 造句、邮件、学术讨论 |


### 2.2 阅读（Reading）—— 约30分钟，自适应

阅读部分包含**三种全新题型**，分布在两个自适应模块中。Module 1表现决定Module 2难度。

#### 题型一：Complete the Words（补全单词）

- **形式**：一段70-100词的学术段落，其中10个单词缺少部分字母，需填入缺失字母补全单词。
- **考查能力**：词汇量、语境推断能力。
- **技巧**：通过上下文语义和语法搭配推测单词，注意词性和时态变化。

#### 题型二：Read in Daily Life（日常阅读）

- **形式**：阅读15-150词的日常短文（邮件、通知、短信、海报、菜单、账单等），回答选择题。
- **考查能力**：主旨理解、细节提取、隐含意义。
- **技巧**：快速扫读，抓关键信息（时间、地点、数字、要求），不需要深度分析。

#### 题型三：Read an Academic Passage（学术阅读）

- **形式**：阅读约200词的短学术文章，回答5道选择题。题材涵盖自然科学、历史、社会科学、艺术等。
- **新增题型**：
  - **Important Idea (Select the Sentence)**：直接在文中点选最重要的句子。
  - **Paragraph Relationship**：分析段落之间的逻辑关系。
- **技巧**：先读题再读文，定位关键信息；注意段落间的因果、对比、递进关系。


### 2.3 听力（Listening）—— 约29分钟，自适应

听力部分包含**四种题型**，涵盖学术和日常场景。同样采用两模块自适应机制。

#### 题型一：Listen and Choose a Response（听句选答）

- **形式**：听一句话（不显示文字），从四个选项中选出合适的回应。
- **考查能力**：语用理解（pragmatics）——理解言外之意。
- **技巧**：注意语调和语气，理解说话人的真实意图。

#### 题型二：Listen to a Conversation（听对话）

- **形式**：听一段校园或日常生活短对话（约10个话轮），回答选择题。
- **考查能力**：对话理解、细节捕捉、推理判断。
- **技巧**：快速记笔记，关注对话中的转折词（but, actually, well）。

#### 题型三：Listen to an Announcement（听通知）

- **形式**：听一段校园或课堂通知（40-85词），回答选择题。
- **考查能力**：信息提取、目的理解。
- **技巧**：重点抓"谁、什么时候、做什么、为什么"。

#### 题型四：Listen to an Academic Talk（听学术讲座）

- **形式**：听一段100-250词的学术讲座，回答选择题。
- **考查能力**：学术听力理解、主旨归纳、细节记忆。
- **技巧**：利用笔记记录关键术语和结构，注意讲座的开头（主题）和结尾（总结）。

**自适应规则**：表现好的考生进入"难模块"（偏学术内容），表现一般的进入"易模块"（偏日常内容）。难模块解锁更高分数上限。


### 2.4 口语（Speaking）—— 约8分钟，非自适应

口语部分从旧版的16分钟缩短至8分钟，**取消准备时间**，共11题。

#### 题型一：Listen and Repeat（听句复述）—— 7题

- **形式**：听一系列难度递增的句子（校园/日常话题），逐句复述。
- **考查能力**：发音、语调、听觉记忆。
- **技巧**：集中注意力，尽可能完整准确地重复原句；句子会越来越长，提前适应。

#### 题型二：Take an Interview（即兴面试）—— 4题

- **形式**：围绕一个话题回答4个问题，每题45秒作答时间，**无准备时间**，问题难度递增。
- **考查能力**：即兴表达、逻辑组织、流利度。
- **技巧**：
  - 采用**"观点+理由+例子"**结构快速组织回答。
  - 不要沉默——即使没想好也要先说，边说边组织。
  - 45秒不需要说太多，2-3个完整句子即可。

**评分方式**：Listen and Repeat由AI机器评分；Take an Interview采用人机混合评分，评估流利度、连贯性、语法和发音。


### 2.5 写作（Writing）—— 约23分钟，非自适应

写作部分**取消了综合写作和独立议论文**，替换为三种新任务。

#### 题型一：Build a Sentence（造句重组）—— 10题，约7分钟

- **形式**：看到10个句子，每个句子有一个回复，但回复的词序被打乱，需重新排列为正确语序。
- **考查能力**：语法结构、句法知识。
- **技巧**：先找主语和动词，确定句子骨架，再填入修饰成分。

#### 题型二：Write an Email（写邮件）—— 1篇，约7分钟

- **形式**：根据给定场景写一封邮件。
- **考查能力**：实用写作、语域（正式/非正式）把握、信息组织。
- **技巧**：
  - 开头明确目的，中间展开细节，结尾礼貌收束。
  - 注意邮件格式和语气（根据场景判断正式或非正式）。

#### 题型三：Writing for an Academic Discussion（学术讨论写作）—— 1篇，约10分钟

- **形式**：在课堂讨论板上就某个话题发表观点（延续自旧版托福同名题型）。
- **考查能力**：学术写作、论点表达、逻辑论证。
- **技巧**：
  - 先表明立场，再给出2个支撑理由。
  - 可以引用或回应讨论中其他人的观点来增加互动性。
  - 10分钟写100-150词即可，不需要写长篇大论。

**评分方式**：Build a Sentence为机器评分（比对标准答案）；Write an Email和Academic Discussion由AI评分引擎评估，考查任务完成度、连贯性、词汇资源和语法准确性。

## 四、自适应机制详解

2026新托福的阅读和听力采用**多阶段自适应测试（MST）**：

1. **Module 1**：所有考生难度相同，是"定位模块"。
2. **Module 2**：根据Module 1表现分流——
   - 表现优秀 → 进入**难模块**（偏学术内容），最高可达6.0分。
   - 表现一般 → 进入**易模块**（偏日常内容），最高封顶4.0分。

**关键策略**：Module 1的正确率直接决定分数天花板。必须全力攻克第一模块，确保基础题正确率在85%以上。


## 五、分阶段备考方法

建议备考周期**2-4个月**（取决于英语基础）。

### 第一阶段：熟悉新题型（第1-2周）

- 仔细阅读ETS官方对新题型的说明，做ETS提供的样题。
- 做1-2套完整模考，熟悉100分钟的考试节奏和自适应体验。
- 评估自己的薄弱环节（词汇？听力？口语反应速度？）。

### 第二阶段：基础强化期（第3-6周）

**词汇**：
- 每天30分钟背诵托福核心词汇。Complete the Words题型对词汇量要求极高，需要掌握**拼写**而不仅仅是认识。
- 推荐使用词根词缀法，同时结合语境记忆。

**听力**：
- 每天精听1-2段学术讲座和日常对话，训练笔记能力。
- Listen and Choose a Response题型需要专项训练语用理解——理解言外之意。
- 适应不同口音（美音、英音、澳洲口音等）。

**口语**：
- 每天练习Listen and Repeat——找TED Talks或VOA短句逐句复述。
- 每天用"观点+理由+例子"框架练习即兴回答，限时45秒。

**写作**：
- 练习Build a Sentence——做语法句式练习，熟悉各类从句、倒装、虚拟语气。
- 每周写2-3封模拟邮件，注意正式/非正式语域区分。

### 第三阶段：专项突破期（第7-10周）

- **阅读**：限时做Complete the Words和日常阅读题，训练30-45秒/题的做题节奏。
- **听力**：对标考试难度做精听，重点攻克学术讲座题型。
- **口语**：录音回听，纠正发音和语调问题；模拟面试环节练无准备时间的即兴回答。
- **写作**：Academic Discussion限时10分钟写完，练习快速构思和论证。

### 第四阶段：冲刺模考期（最后2周）

- 每周做2-3套完整模考，严格计时。
- 重点分析Module 1的正确率——目标85%以上。
- 回顾错题，总结高频考点和易错陷阱。
- 调整考试节奏，适应无休息的100分钟连续作答。

## 七、自适应机考特别提醒

1. **"开局即决战"**——阅读和听力的Module 1表现直接决定Module 2的难度和你的分数上限。易模块最高只能拿4.0。
2. **时间分配**——100分钟无休息，口语和写作在后半段，保持体力和专注力至关重要。
3. **旧版刷题策略大幅失效**——旧版长篇阅读和综合写作的练习对新题型帮助有限。必须使用2026年新题型材料备考。
4. **词汇拼写能力**——Complete the Words要求你拼出单词，仅"认识"不够，必须练习拼写。
5. **口语无准备时间**——Take an Interview的4道题没有准备时间，需要极强的即兴反应能力，平时多做限时口语练习。
6. **AI评分已大规模应用**——口语和写作部分采用AI评分，模板化答案可能不再有效，要注重真实表达能力。
